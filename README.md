# 🛡️ FAILSAFE - Student Early Warning System

An End-to-End Explainable AI (XAI) dashboard designed to detect academic drop-out risks early, built using **FastAPI**, **XGBoost**, and **SHAP**.

## 🚀 Key Features
* **Machine Learning Powered:** Uses optimized XGBoost to classify student academic safety.
* **Explainable AI (XAI):** Uses live SHAP values to dynamically visualize feature impact for every specific student directly on the frontend.
* **Automated Intervention Logic:** Provides personalized recovery strategies depending on academic weak spots.
* **Modern Interface:** Sleek, dark-themed responsive frontend crafted with Tailwind CSS.

---

## 🛠️ Tech Stack & Architecture
* **Frontend:** HTML5, Tailwind CSS, Native JavaScript (Fetch API)
* **Backend:** FastAPI (Python), Uvicorn Server
* **ML Pipeline:** Pandas, Scikit-Learn, XGBoost Classifier, Joblib
* **Interpretability:** SHAP (Shapley Additive exPlanations)
